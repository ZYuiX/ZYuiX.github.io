LeetCode.104 二叉树的最大深度

理解概念:

​ 二叉树的深度：从根节点出发，到叶子结点。（根结点深度为1，叶子节点高度为n）

​ 二叉树的高度: 从叶子结点出发，到根节点。（叶子结点高度为1，根节点高度为n）

本题求最大深度，可以与最高度等效；

题解：递归法(后续遍历)

​ 1.确定递归函数的参数和返回值

​ int getheight(TreeNode root)

​ 2.确定终止条件

​ if(root==null) return

​ 3.单层递归的逻辑

​ 3.1 求左子树的高度

​ leftheight = getheight(root.left) 左

​ rightheight = getheight(root.right) 右

​ height = 1 +max(leftheight,rightheight)//最后要加上根节点的高度 中

​ 3.1中先求左子树高度，再求右子树高度，最后加上根节点的高度。满足左中右的遍历顺序。利用的后序遍历

题解:层序遍历(每遍历一层，深度加1,效率很低)

层序遍历回顾：层序遍历需要引入辅助队列，深度优先遍历需要引入栈。

LeetCode 111.二叉树的最小深度

注意点：当根节点的左子树为空时，最小深度并不是1，而是右子树中寻找，反之亦然；

​ 1.如果左子树为空且右子树不为空，那么最小深度应该根节点加上右子树的最小深度

​ 2.如果右子树为空且左子树不为空，那么最小深度应该是根节点加上左子树的最小深度

​ 3.如果左右子树都不为空，那么就需要比较左右字数的最小深度了。

递归法解决

层序遍历法解决：层序遍历，深度逐层增加，当某一节点的左右孩子都为null，说明是叶子结点，此时就是最小深度。
